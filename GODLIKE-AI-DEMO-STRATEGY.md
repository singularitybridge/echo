## üé¨ Production Strategy

### Critical Success Factor: VARIETY

Each scene MUST look completely different:
- Different character appearance
- Different clothing/costume
- Different setting/background
- Different lighting mood
- Different action/movement

**This variety proves the AI's range and power**

### Reference Mode Strategy

**All scenes use referenceMode: 1** (no shot continuity)

**Why?**
- We WANT each scene to be completely different
- No need for continuity between scenes
- Each scene stands alone as proof of capability
- Maximum diversity = maximum impact

### Simple Actions per Scene

Keep actions minimal and achievable:
- Look around ‚Üí Simple head movement
- Point/Gesture ‚Üí Simple arm movement
- Draw sword ‚Üí Simple prop interaction
- Walk forward ‚Üí Simple locomotion
- Chop vegetables ‚Üí Simple hand action
- Stretch ‚Üí Simple body movement
- Paint ‚Üí Simple tool interaction

**Why simple?**
- AI handles these actions reliably
- Focus is on VARIETY not complexity
- Each scene needs to succeed on first try
- 8 scenes = 8 different successes needed

## üéØ Character Reference Strategy

### Option 1: Single Reference per Scene (Recommended)

**Create 7 different character references:**
1. Futuristic robot (metallic, high-tech)
2. Medieval knight (armor, helmet, sword)
3. Business person (suit, professional)
4. Ocean diver (wetsuit, diving gear)
5. Professional chef (white uniform, chef hat)
6. Athletic runner (sportswear, running shoes)
7. Artist painter (casual clothes, paint splattered)

**Pros:**
- Maximum variety and differentiation
- Each character clearly distinct
- Shows AI can handle any character type

**Cons:**
- More upfront work (7 reference sets)
- More cost (~$175 vs $25)

### Option 2: Text-to-Video Only (Faster, Riskier)

**No character references, rely on text prompts alone**

**Pros:**
- Fastest to execute
- Lowest cost
- Tests pure text-to-video capability

**Cons:**
- Less control over character appearance
- Higher chance of inconsistency within each scene
- Riskier for investor demo

### Recommended: Option 1

**Why:** You get ONE shot at impressing investors. The $150 difference in reference generation is insignificant compared to closing a funding round.
